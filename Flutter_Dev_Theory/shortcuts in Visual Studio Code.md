Here are some commonly used shortcuts in Visual Studio Code:

### General

- **Command Palette**: `Ctrl+Shift+P` or `F1`
- **Quick Open**: `Ctrl+P`
- **New Window**: `Ctrl+Shift+N`
- **Close Window**: `Ctrl+Shift+W`

### Basic Editing

- **Cut line (empty selection)**: `Ctrl+X`
- **Copy line (empty selection)**: `Ctrl+C`
- **Move line up/down**: `Alt+Up` / `Alt+Down`
- **Copy line up/down**: `Shift+Alt+Up` / `Shift+Alt+Down`
- **Delete line**: `Ctrl+Shift+K`
- **Insert line below**: `Ctrl+Enter`
- **Insert line above**: `Ctrl+Shift+Enter`
- **Go to beginning of line**: `Home`
- **Go to end of line**: `End`
- **Go to line...**: `Ctrl+G`
- **Go to file...**: `Ctrl+P`
- **Find**: `Ctrl+F`
- **Replace**: `Ctrl+H`
- **Find next/previous**: `F3` / `Shift+F3`

### Multi-cursor and Selection

- **Add cursor above/below**: `Ctrl+Alt+Up` / `Ctrl+Alt+Down`
- **Undo last cursor operation**: `Ctrl+U`
- **Insert cursor at end of each line selected**: `Shift+Alt+I`
- **Select all occurrences of current selection**: `Ctrl+Shift+L`
- **Select all occurrences of current word**: `Ctrl+F2`

### Rich Languages Editing

- **Trigger suggestion**: `Ctrl+Space`
- **Trigger parameter hints**: `Ctrl+Shift+Space`
- **Format document**: `Shift+Alt+F`
- **Format selection**: `Ctrl+K Ctrl+F`
- **Go to definition**: `F12`
- **Peek definition**: `Alt+F12`
- **Go to implementation**: `Ctrl+F12`
- **Quick fix**: `Ctrl+.`

### Navigation

- **Show Explorer/Toggle Focus**: `Ctrl+Shift+E`
- **Show Search**: `Ctrl+Shift+F`
- **Show Source Control**: `Ctrl+Shift+G`
- **Show Extensions**: `Ctrl+Shift+X`
- **Show Debug**: `Ctrl+Shift+D`
- **Show Output**: `Ctrl+Shift+U`

### Debug

- **Start/Continue**: `F5`
- **Stop**: `Shift+F5`
- **Step into**: `F11`
- **Step over**: `F10`
- **Step out**: `Shift+F11`
- **Toggle breakpoint**: `F9`
- **Show Debug Console**: `Ctrl+Shift+Y`

### Integrated Terminal

- **Show integrated terminal**: `Ctrl+``
- **Create new terminal**: `Ctrl+Shift+``
- **Focus next/previous terminal**: `Ctrl+PageDown` / `Ctrl+PageUp`

These shortcuts cover a wide range of functionality in Visual Studio Code, helping you to code more efficiently.




To comment and uncomment code in Visual Studio Code, you can use the following shortcuts:

### Single Line Comment

* **Windows/Linux** : `Ctrl+/`
